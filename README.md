# Packet Quick Share GNOME Extension

GNOME Shell extension which integrates [Packet] with the Quick Settings panel.

[Packet] allows you to use Google/Android's [Quick Share] protocol for transferring files between devices. This shell extension adds a GNOME Quick Settings entry to control [Packet] without opening the app.

[Packet]: https://github.com/nozwock/packet
[Quick Share]: https://www.android.com/better-together/quick-share-app/
